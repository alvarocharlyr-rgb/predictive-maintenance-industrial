# 🔧 Predictive Maintenance Industrial

## 📊 Descripción

Este proyecto desarrolla un modelo de Machine Learning para predecir fallas en maquinaria industrial utilizando datos de sensores.

## 🎯 Objetivo

Detectar condiciones que anticipan fallas y permitir mantenimiento predictivo.

## 🧠 Metodología

* Análisis exploratorio de datos (EDA)
* Identificación de outliers
* Modelado con Random Forest
* Evaluación con métricas de clasificación

## 📈 Resultados

* Se logró predecir fallas con alta precisión
* Variables clave: Torque, Temperatura y Velocidad
* Mejora en detección de fallas mediante balanceo de clases

## ⚙️ Tecnologías

* Python
* Pandas
* Scikit-learn
* Matplotlib / Seaborn

## 📂 Dataset

Descargar desde:
https://www.kaggle.com/datasets/shivamb/machine-predictive-maintenance-classification

Colocar el archivo en:
`data/predictive_maintenance.csv`


## 💡 Conclusión

El uso de Machine Learning permite anticipar fallas en equipos industriales, reduciendo costos y mejorando la eficiencia operativa.
